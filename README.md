# MotionSwing — AI Motion Controller 🎮

AI-powered motion detection game playable on mobile and tablet devices.

## 🎯 How to Play

1. Open the game link on your mobile/tablet
2. Click **"เปิดกล้อง / START"** to enable camera
3. Use a bright colored object (ball, cloth, or marker)
4. Click **"LOCK COLOR"** to calibrate
5. Swing the object quickly to score!

## 📱 Play Now

- **GitHub Pages:** https://yumaHD.github.io/MotionSwing/
- **Direct HTML:** Open `index.html` in browser

## ✨ Features

- Real-time motion detection
- 5 color themes
- 3-minute gameplay
- Score tracking with history
- Mobile & tablet optimized
- No downloads required

## 🛠️ Technical Stack

- Pure HTML5 + Canvas
- WebGL Camera API
- CSS3 Animations
- LocalStorage for history

Enjoy! 🚀
